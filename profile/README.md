# 🦇 Batear: The Open-Source Drone Detection Ecosystem

**Batear** is a high-performance, community-driven framework designed to detect and track drone activity using affordable, off-the-shelf hardware.

[**Explore the Firmware**](https://github.com/batear-io/batear) | [**Get Involved**](https://github.com/batear-io/batear/issues)

### 💡 The Batear Advantage: Acoustic Defense for Everyone

Batear is built on **Radical Accessibility**. We believe airspace security shouldn't be a luxury.

* **Ultra-Low Cost**: By using ESP32 and I2S microphones, we reduce detection costs from thousands to **under $30 USD**.
* **Plug-and-Play**: Batear is designed for easy, widespread deployment by anyone, anywhere.
